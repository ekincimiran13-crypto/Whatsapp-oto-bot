# WhatsApp Otomatik Mesaj Botu

Bu bot, WhatsApp üzerinden komutlarla kontrol edilebilen ve belirlediğiniz gruplara her 3 dakikada bir mesaj gönderen bir uygulamadır.

## Komutlar
- `.ekle` : Botun olduğu grupta bu komutu yazın, o grubu listeye ekler.
- `.baslat [mesaj]` : Otomatik mesajı başlatır. Örnek: `.baslat Merhaba!`
- `.durdur` : Mesaj gönderimini durdurur.
- `.liste` : Kaç grubun ekli olduğunu gösterir.

## Kurulum
1. GitHub deponuzu Render veya Railway'e bağlayın.
2. Log (Terminal) kısmında çıkan QR kodu WhatsApp'tan taratın.
3. Bot hazır!
